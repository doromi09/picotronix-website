Smart Farm

스마트 농업 솔루션

개요

IoT 기술을 활용한 스마트팜 측정 및 제어 시스템을 개발합니다. 데이터 기반의 정밀 농업을 실현합니다.

Picotronix의 스마트팜 솔루션은 농작물 생장에 필요한 환경을 최적화하고, 실시간 모니터링과 자동 제어를 통해 생산성을 향상시킵니다. 농업의 디지털 전환을 선도합니다.

제품 및 서비스

토양상태 측정기

토양 수분, pH, EC(전기전도도), 온도 등을 실시간으로 모니터링합니다. 정확한 센싱 기술로 토양 상태를 파악하여 최적의 재배 환경을 만듭니다.

배지상태 측정기

수경재배 배지의 양분, EC, pH 등을 측정하여 식물에 필요한 영양분을 정확히 공급합니다. 배양액의 상태를 지속적으로 모니터링합니다.

생장 모니터링

식물의 생장 데이터를 수집하고 분석하여 생장 패턴을 파악합니다. 영상 기반 모니터링으로 식물의 상태를 정밀하게 관찰합니다.

자동 제어

측정된 환경 데이터를 기반으로 관수, 양액 공급, 온도, 습도, 조명 등을 자동으로 제어합니다. 최적의 생장 환경을 24시간 유지합니다.
